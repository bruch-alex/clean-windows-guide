# Установка

## Создание локального аккаунта и пропуск аккаунта microsoft

После выбора страны вам нужно:

1. На экране выбора сети wifi нужно открыть командную строку нажатием `sfift + f10`
2. Написать `OOBE\BYPASSNRO` 
3. Система перезагрузится и начнется оффлайн установка
4. Далее вы сможете создать локальный аккаунт
